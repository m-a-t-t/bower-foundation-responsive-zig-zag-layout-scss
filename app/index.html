Look for "zig-zag-titles" in source to see how it works

When you add "zig-zag-titles" class to parent element its "content-w" class children will push and pull "content-header" and "content-body" elements. You can disable zig-zag on any "content-w" element by adding "no-zig-zag" class. Or you can force title to the left or right by adding "zig-zag-left" or "zig-zag-right" to "content-w" element.

Basic structure of elements:

<div class="content-w"> <section class="row content "> <header class="content-header"> <i class="fa-magnet fa fa-4x double-border icon-center"></i> <h1 class="content-title">Title 1</h1> </header> <div class="content-body"> <p> And main text gows here... </p> </div> </section> </div>

There's also "overlapped-items" class added so icon overlaps previous element.
